# BimmerHub

Mini CMS postavený na Next.js 16 (App Router). Aplikace kombinuje veřejnou magazínovou část o BMW, interní dashboard pro správu vlastních článků a vlastní API nad Prisma ORM.

**Live aplikace:** [https://2025-p4a-minicms-deimlinghoracek.vercel.app/](https://2025-p4a-minicms-deimlinghoracek.vercel.app/)

Zdrojový kód aplikace je v adresáři [`bimmerhub/`](./bimmerhub). Podrobná dokumentace, návod ke spuštění a deployment guide jsou v [`bimmerhub/README.md`](./bimmerhub/README.md).
